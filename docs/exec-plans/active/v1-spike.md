# Alpha Plan

## Objective

Keep the desktop panel useful for Codex issue-learning sessions.

## Steps

1. Keep Electron app shell stable.
2. Connect to `codex app-server`.
3. Resume and optionally follow the latest local thread.
4. Reduce thread items into route/status state.
5. Persist state and knowledge notes locally.
6. Display always-on-top route/status UI.
7. Keep automated smoke checks passing.

## Done When

`npm run doctor` passes, and the packaged app launches locally.
